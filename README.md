# Mirror_Draw_pad-Python
Pygame symmetric Pixel pattern draw pad game. 
## Keys
q = QUIT
d = UP
f = RIGHT
s = LEFT
e = UP
c = +BRUSH
v = -BRUSH
r = RESET
